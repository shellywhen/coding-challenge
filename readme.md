## Local Setup

### Dependency

- MongoDb
- Python 3+
- npm

### Backend

```sh
cd code/backend
pip3 install -r requirements.txt
python3 run.py
```

The server will be run at `localhost:12050`.

It will use MongoDb at `localhost:27017`.

### Frontend

```sh
cd code/frontend
npm install
npm run serve
```
